raspi-connect

发表于 2017-03-18 |

最近做毕设用的树莓派0，总结一下usb网卡和wifi 8188eu安装过程

事先准备

  • 一个树莓派
  • 一张sd卡,以及读卡器
  • mini micro 公 转 hdmi/vga 母 线 (连接显示屏)
  • OTG 转 usb 供电线 (以前的安卓线即可)
  • OTG 转 USB-HUB
  • usb键盘,显示屏,usb网卡/无线网卡

安装系统及配置网络

  1. 官网上下raspbian系统,pixel带GUI。
  2. SDFormatter,Win32DiskImager分别格式化以及写系统
  3. 启动 登录 pi;raspberry
  4. sudo raspi-config interfaces那一栏enable SSH
  5. USB网卡一边用校园网一边连到usb-hub。参考http://blog.csdn.net/cxhome213/article/details/8983247
    感觉没啥用，我是树莓派重启之后连到显示的ip
  6. 因为我买的无线网卡树莓派不能识别，要装驱动，参考
    https://www.raspberrypi.org/forums/viewtopic.php?p=462982#p462982
    我的系统是4.4.50+ #970 则
    wget http://www.fars-robotics.net/8188eu-4.4.50-970.tar.gz
    输入命令后出错 no data received 因此用windows下载上述压缩包通过filezilla传到树莓派，解压缩，添加权限chmod +x install.sh 之后./install.sh运行
    装好驱动之后
    sudo nano /etc/network/interfaces
    修改wifi设置
    auto wlan0
    allow-hotplug wlan0
    iface wlan0 inet dhcp
    wireless-essid name_of_wifi_without””
    设置密码
    sudo nano /etc/wpa_supplicant/wpa_supplicant.conf
    添加
    network={
    ssid = “name_of_wifi”
    psk=”wifi_password”
    }
    重启或者刷新网络就可以了，路由器dhcp绑定ip和mac地址就可以通过putty不需要显示屏就可以连接了

    配置多个wifi网络

  • 首先在etc/network/interfaces 新增wlan，照着wlan0写就可以
  • 在wpa_supplicant.conf添加多个network元素，分别填写ssid和psk，添加priority属性，属性值为数字，数字越大越优先连接 比如我设置了1，3，5三个wifi的priority属性值

    遇到的问题

  • usb网卡无法通过笔记本分享
  • netsh wlan show profile name=”Profile-Name” key=clear 获取wifi密码
  • 少看中文资料
